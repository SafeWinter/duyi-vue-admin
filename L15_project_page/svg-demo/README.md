# 示例说明



本示例用于实测 `require.context()` 函数在 `SvgIcon` 组件中的应用，并在 `DeepSeek` 的提示下基于 `JS` 原生语法实现了 `SVG` 元素的手动渲染。

使用方法：

```bash
node -v
v14.21.3
# 安装依赖
npm i
# Webpack 打包
npm run build
# 用 Live Server 打开 dist 中的打包结果
```
